# Feather/Fur shader for VRChat
A fork of [NormalizedCrow's Feather/Fur shader](https://normalizedcrow.gumroad.com/l/FeathersAndFur) with a few small tweaks and extra features.

> [!IMPORTANT]
> This repository does not include paid features like the coat painting tool or clothing mask editor. If you want to use them, you must buy the [premium version from NormalizedCrow](https://normalizedcrow.gumroad.com/l/FeathersAndFur). <ins>***Only import the `Premium` folder!***</ins>

### Documentation
See the original shader's documentation [here](https://www.youtube.com/watch?v=kvFP_khHFrk&list=PL8qseEavOSUAz-Lj4z-7nzOWb5K6y5jb_), and its clothing mask editor docs [here](https://docs.google.com/document/d/1r3heqsGQbVG4nRyRLbYfdFNrJO2N1XiQ03PzJ1e5zUY).

### What's different
- Ported to VCC package
- Additional card textures
- Light volumes support
- Specular antialiasing and specular highlights from light probes
- Optimizations
- Optional audiolink mode for the Fluffy Gryphon base
